## `x`: тэги
`xy`/`xy=z`
- [ ] `@x:string!` чек класса
- [ ] `$x:number!` чек количества детей
- [ ] `#x:string!` чек имени
- [ ] `->x:string!` сохранить результат в `getgenv()[x]`
- [ ] `}x:string!` сохранить шаблон
- [ ] `{x:string!` загрузить шаблон
- [ ] `'x:string!` чек атрибута
## `w?`: модификаторы
- [ ] `!` not
- [ ] `|` or
- [ ] `&` and
- [ ] `+` добавление
- [ ] `-` исключение
- [ ] `~` нечёткость
- [ ] `?` 
### `f?`: модификаторы для чисел
- `>y` больше чем `y`
- `<y` меньше чем `y`
- `>=y` больше или равно `y`
- `<=y` меньше или равно `y`
## `q?`: структура
- [ ] `;` вывод n+ gins
- [ ] `:` компиляция фильтра для повторных использований <u>сделать глобалку ins чтобы можно было компилировать универсальные фильтры</u>
- [ ] `\` экранирования символа
- [ ] `,` пайплайн
